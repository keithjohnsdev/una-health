You down with EVP?
Yeah you know me!

UNA Test uses EVP - a content validation methodology leveraging subject matter experts to identify critical aspects of job duties, knowledge, skills, and abilities required of the specialty, allowing test items to be directly linked to clinical practice requirements.

EVP methodology is superior to other techniques because:

  • Test items are not simply recall of basic information.
  • Analysis and synthenization of content is incorporated into each test item.
  • High-fidelity patient care scenarios parallel actual job experiences.
  • Subject matter experts decide difficulty of each item.
  • Test content incorporates critical thinking skills.
  • Each test is supported by a professional technical report aggregating all EVP data, supporting use for pre-hire screening.

The following tests have been validated using EVP:

  • RN Pharmacology
  • Medical Surgical RN
  • Intensive Care Unit RN
  • Emergency Department RN
  • Operating Room RN
  • Post Anesthesia Care Unit RN
  • Telemetry RN
  • Labor & Delivery RN
  • Physical Therapist
  • Occupational Therapist
  • Speech Language Pathologist
  • Surgical Technologist
  • Certified Nursing Assistant
  • Dysrhythmia Interpretation

...and more are being created today!
